# RPS_Real
A Rock, Paper, Scissors ruby app that allows two players to play and see who won
Written using Ruby 2.1.5p273
Run '$ gem install bundler'
Run '$ gem install rspec'
Run '$ gem install pry'


Launch Sinatra app with '$ ruby app.rb'
